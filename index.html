El conjunto de datos es de Kaggle.El conjunto de datos está relacionado con variantes tintas del vino portugués "Vinho Verde". De calidad media en alcohol, es especialmente apreciado por su frescura.
Las clases están ordenadas y no equilibradas (por ejemplo, hay vinos mucho más normales que excelentes o malos).
Kaggle the White-Wine
Kaggle the Red-Wine
Después de limpiar los dos conjuntos de datos, usando las herramientas apropiadas, los unimos y verificamos con los dos modelos las diferencias entre datos limpios o no limpios.
El conjunto de ambos datos contiene un total de 12 variables, que se registraron para 5037 observaciones.
Estos datos nos permitirán crear diferentes modelos de regresión para determinar cómo diferentes variables independientes ayudan a predecir nuestra variable dependiente, la calidad.
